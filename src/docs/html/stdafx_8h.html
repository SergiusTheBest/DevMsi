Function prototype for LogResult()

Log a message.

If the DLL is being used in a WiX MSI environment, LogResult() will route any log messages to the MSI log file via WcaLog() or WcaLogError().

If the DLL is NOT being used in a WiX MSI environment, LogResult() will route any log messages to stdout or stderr.

If the result is an error code, LogResult will attempt to gather a text version of the error code and place it in the log. For example, if the error code means ERROR_FILE_NOT_FOUND, it will look up the appropriate message ( via FormatMessage() ) and add "The system cannot find the file specified." to the log.

Parameters
hrThe HRESULT to be interrogated for success or failure.
fmtThe string format for a user-specified error message.
