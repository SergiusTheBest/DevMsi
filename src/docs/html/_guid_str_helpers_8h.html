GUID ClassName2GUID ( __in const std::wstring &  ClassName)

Convert a class name to a GUID.

When provided a class name, this function will search the registry to find the GUID associated with the setup class. The search will be case-insensitive: "System" is the same as "system".

An exception will be thrown on error.

Parameters
ClassNameThe setup class name to be searched for (e.g. "system")
Returns
Returns the matching GUID from the system registry.
